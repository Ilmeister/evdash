# evdash – Raspberry Pi EV Dashboard

Moderni, tumma neon -henkinen digitaalinen sähköauton mittaristo
Raspberry Pi:lle. Piirtää 2000×800 resoluutioon (esim. 14" wide-näyttö).

## 1. Vaatimukset

- Raspberry Pi 4 / 5 (suositus)
- Raspberry Pi OS (Bookworm / Bullseye)
- 14" näyttö (2000×800) HDMI/DSI
- CAN-liityntä (esim. MCP2515-pohjainen HAT tai sisäinen CAN)
